mybrary

mybrary est une application destinée aux bibliothécaires. Elle permet d'ajouter des livres à votre bibliothèque en ligne pour les retrouver facilement.
Parmi les options présentes, on peut ajouter la couverture du livre ainsi que le nombre de pages.

live-chat

live-chat est une application de chat en ligne. Elle permet de discuter avec les personnes présentes et de se retrouver en amis.
Parmi les options présentes, on peut créer un salon privé avec mot de passe.

  • Voir sur GitHub

restful_api

restful_api est une application écrite en GO. Elle permet de lister les utilisateurs et d'en créer.
Parmi les options présentes, on peut créer des utilisateurs, les supprimer et afficher une liste de ces-derniers.
